> **Disclaimer:** If yer song suggestion doesn’t make us laugh, cry, or question our life choices, it might not make the cut. No hard feelings—just like `pacman`, we’re picky.

---

### **How to Add a Song**
1. Fork this repo.
2. Edit this file (`songs.md`) and add yer suggestion to the **Current Requests** table below.
3. Submit a Pull Request (PR) with:
   - **Title:** `"Added [Song Name] by [Artist] for [Category]"`
   - **Description:** Explain why this song belongs in the playlist (bonus points for humor).
4. Wait for the maintainer to merge it, reject it, or ignore it while sobbing softly.
